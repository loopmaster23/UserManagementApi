# User Management API (.NET 7)

This project is a RESTful Web API built with ASP.NET Core.

## Endpoints

- `GET /api/users`
- `POST /api/users`
- `GET /api/users/{id}`
- `PUT /api/users/{id}`
- `DELETE /api/users/{id}`

## Features

- Validation
- Middleware logging
- Developed and debugged using GitHub Copilot
